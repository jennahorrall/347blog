Modals

This week I explored modal boxes. I didn't realize how often modal boxes are used today, but I see them quite frequently as login/sign up boxes when visiting a website. Modal boxes extend to cases beyond that as well because they highlight important information that the viewer needs to see right away. According to this blog, there are five use cases that modal boxes are commonly used for:

  • Alert the user that an error has occured
  • Warn the user of something
  • Collect data from users
  • Remind user of something before moving on
  • Inform users of important information

There's quite a lot happening in the background to display this modal box. The button that opens the modal triggers the modal box to display on the screen. The modal is styled in CSS, but it's display property is set to none so that it doesn't display until the button is clicked. Once the button is clicked, the script grabs the modal element and sets it's display to block. I thought I would have to do something fancy to make the background fade out, but the background color of the element is just set to black with a greater opacity level.

One new property tag that I hadn't seen before today was the z-index tag. This is set to 1 for the modal element. This tag essentially moves the element to be at the top of the display stack. It only works with positioned elements, and because the modal element has a fixed position, the element appears to be "on top of" the other elements.

I also learned something new about JavaScript by making this modal box. Instead of including the function inside of the button element using the onclick function, the demo calls onclick in the script directly:

                
                    btn.onclick = function() {
                        modal.style.display = "block";
                    }

I personally think this is cleaner and more concise because the HTML code has content, JavaScript code has functionality, and CSS has styling. Each component only has code pertaining to it's sole function.

Takeaways

Creating a modal box was very fun and I enjoyed learning about them. They have many uses and it was a lot easier to implement then I had expected.
