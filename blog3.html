Printing in JavaScript

This week I decided it was best to look into JavaScript techniques as we will be learning more about it soon. I came across this cool video below containing some 'hacks' you can do when printing to the console in JavaScript. These will definitely be helpful when we start learning about JavaScript and do testing/debugging.

Prior to watching this video, I wanted to make sure I was clear on the many different ways to print something in JavaScript. Following this w3schools documentation, I learned that there are actually four different ways to print something in JavaScript:

  • innerHTML: This method prints html elements using JavaScript. To use this, simply create a script that references the element and call ".innerHTML" on that element. This method is more helpful if you need to know the value of an html element dynamically.
  • document.write(): This is similar to innerHTML, but if it's used after the html page is loaded then it will overwrite all of the HTML. It's recommended that this method should only be used for testing.
  • window.alert(): You can place window.alert() in a script and pass whatever you want to print as an argument. I think the drawback of this method is that it comes up as an alert box, so it might get annoying unless you want to see it immediately.
  • console.log(): The final method that is also referenced in the video is console.log(). This will display in the browser and is mainly used for debugging purposes. In order to see it, you have to open up your browsers developer tools and go to the console to see it.

This video below talks about two "hacks" when printing to the console.

Time Log

Time log comes in handy when you are trying to measure performance of a function or a portion of code. If you want to time how long something takes to run, you can initialize a timer using: 

console.time('description');
To print the time at a certain part within your code: 
console.timeLog('description');
To end the timer and print the final result: 
console.timeEnd('description');

Table Log

The next console hack is printing an array to the console in the form of a table. If you have an array that has consistent data attributes, you can print it to the console as a table: 

console.table(array);
For instance, if you have an array named 'color' with two elements: each containing 'name' and 'favorite color,' you can print that array to the console: 
console.table(color);
The result will be a nicely formatted table that displays in the console.

Takeaways

I enjoyed learning about this topic and it's interesting to me that there are so many ways to print one value using JavaScript. It increases the flexibility of the language.
