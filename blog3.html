Console Hacks Using JavaScript

This week I decided it was best to look into JavaScript techniques as we will be learning more about it soon. I came across this cool video containing some 'hacks' you can do when printing to the console in JavaScript.

If you are debugging some code and want to quickly see the value of a variable, these suggestions make it easier to view those things when printing to the console.

Time Log

Time log is very helpful when you are trying to measure performance of a function or a portion of code. If you want to time how long something takes to run, you can initialize a timer using: 

console.time('description');
To print the time at a certain part within your code: 
console.timeLog('description');
To end the timer and print the final result: 
console.timeEnd('description');
This funcionality comes in handy when you quickly want to measure the performance of a function.

Table Log

The next console hack is printing an array to the console in the form of a table. If you have an array that has consistent data attributes, you can print it to the console as a table: 

console.table(array);
For instance, if you have an array named 'color' with two elements: each containing 'name' and 'favorite color,' you can print that array to the console: 
console.table(color);
The result will be a nicely formatted table.
